

Welcome to zasm
---------------
 
zasm is a 2-pass assembler for the Zilog Z80 cpu.
it's a command line assembler for unix-style platforms.

	- binary and intel-hex output files
	- eprom and ram targets
	- misc. ZX Spectrum file types

Installation
------------

Simply extract the archive and install the binary into "/usr/bin/":

	1. double-click the archive
	2. open a terminal window                       --> Applications/Utilities/Terminal
	   type:   cd <path to extracted folder>
	   type:   sudo cp zasm /usr/bin				<-- install program
	   type:   zasm test/*.ass						<-- run test(s)
